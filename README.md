# Oversight
Yet another Sandbox Game with a twist

Current Controls (with corresponding actions):
Escape - Closes Application
F11 - Switch Fullscreen and Window Mode

Can receive (but no output)
- Mouse Scroll
- Mouse Click
- Keyboard Press
